**Highlights:** Auto Scaling with EC2 + ALB + self-healing ASGs · Redis sub-ms caching · CloudFront CDN (sub-20ms global latency) · rate limiting & token blacklisting · CloudWatch + SNS alerting · companion React Native (Expo) app
**Tech:** AWS, Redis, CloudFront, React Native, Node.js
**Status:** ✅ Completed

### ☎️ AI Voice Receptionist — Multi-Tenant Hotel Automation
Production-grade voice agent powered by **vapi.ai** for natural, real-time phone conversations.
**Highlights:** split-path architecture — a WebSocket stream handles sub-second STT/TTS + reasoning, while a REST layer runs synchronous DB tools · atomic MongoDB operations (`check_availability`, `book_room`, `cancel_room`) preventing double-booking under load
**Tech:** vapi.ai, WebSockets, Node.js, MongoDB
**Status:** ✅ Completed

---

### 🖼️ futureFolio — AI-Powered SaaS Portfolio Platform (250+ users)
Lets professionals across photography, dev, design, and writing build AI-analyzed portfolios with an embedded CRM.
**Highlights:** full auth & tiered subscriptions · Lemon Squeezy billing (upgrades/cancellations) · serverless AWS (Lambda + S3) with zero cold-start scaling
**Tech:** Next.js, Express.js, MongoDB, AWS Lambda/S3, Lemon Squeezy, Tailwind
**Status:** 🟢 Live — [snap-folio.com](https://snap-folio.com)
